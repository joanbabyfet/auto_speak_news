## About
基于 Python3 实现自动化朗读柬中时报头条，并保存音频为mp3文件

## Feature

* 采集新闻列表, 网站以 https://cc-times.com 为例
* 引用 pyttsx3 库, 将文本转换为语音, 并控制语速、音量、人声类型
* 朗读新闻并存储文本为音频mp3文件

## Requires
Python 3.11.0  
pyttsx3 2.90
beautifulsoup4 4.11.1  
requests 2.28.1  

## Usage
```
python main.py
```
